# HTML

JavaScript HTML element helpers.

## Install

```
yarn add @arturdoruch/html
````

## Contents

### HTMLElement

HTMLElement object wrapper. Provides common functions to manipulate HTML element.

```js
import HtmlElement from '@arturdoruch/html/lib/element/HtmlElement';
```
